# MindMap

MindMap turns outlines and structured ideas into interactive mind maps inside Cursor.

The plugin ships as a single Cursor plugin and points Cursor at the hosted `MindMap` MCP endpoint.

## Install

Install `MindMap` from the Cursor Marketplace once the listing is approved.

## What The Plugin Includes

- `.cursor-plugin/plugin.json` for the marketplace manifest
- `mcp.json` with the hosted MCP server configuration
- `assets/logo_mindmap.svg` for marketplace branding

## Hosted MCP Server

The plugin connects Cursor to:

`https://mindmap-cursor.keenethics-labs.com/mcp`

Authentication is currently not required.
